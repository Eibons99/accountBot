# Telegram 记账机器人 - 指令使用手册

## 📱 快速上手

### 新手入门三步曲
1. **启动机器人**：发送 `/start`
2. **查看帮助**：发送 `/help`
3. **开始记账**：直接发送 `100 早餐`

---

## 🎯 基础指令

### 1. 系统指令

| 指令 | 说明 | 权限 |
|------|------|------|
| `/start` | 启动机器人，显示欢迎信息 | 所有人 |
| `/help` | 显示帮助信息和指令列表 | 所有人 |
| `/ping` | 测试机器人是否在线 | 所有人 |

### 2. 记账指令

#### 个人记账（操作员）
```
金额 备注
```
**示例：**
```
100 早餐
50 地铁
200 超市购物
```

#### 群组记账（标记员）
```
@成员 金额 备注
```
**示例：**
```
@张三 100 午餐
@李四 50 奶茶
@王五 200 电影票
```

#### 多笔连续记账
```
100 早餐
@张三 80 午餐
50 交通
@李四 30 咖啡
```

## 👥 人员管理指令

### 查看人员列表

#### 查看所有标记员
```
/标记员列表
```
或
```
/查看标记员
```

**输出示例：**
```
标记人员列表（3 人）：
张三
李四
王五
```

#### 查看所有操作员
```
/操作员列表
```
或
```
/查看操作员
```

**输出示例：**
```
操作员列表（2 人）：
张三
赵六
```

### 设置人员权限

#### 设置标记员
```
设置标记员@用户名
```
或
```
设置标记人@用户名
```

**示例：**
```
设置标记员@张三
```

**权限说明：**
- 标记员可以使用 `@成员 金额 备注` 格式记账
- 需要管理员权限才能设置

#### 设置操作员
```
设置操作员@用户名
```
或
```
设置操作人@用户名
```

**示例：**
```
设置操作员@赵六
```

**权限说明：**
- 操作员可以直接发送 `金额 备注` 进行记账
- 操作员可以确认入账/出账
- 需要管理员权限才能设置

### 取消人员权限

#### 取消标记员
```
取消标记员@用户名
```
或
```
取消标记人@用户名
```

**示例：**
```
取消标记员@张三
```

#### 取消操作员
```
取消操作员@用户名
```
或
```
取消操作人@用户名
```

**示例：**
```
取消操作员@赵六
```

## 📊 查询统计指令

### 今日统计
```
/today
```
显示今日收支汇总、分类统计。

**功能特性：**
- ✅ 点击入款金额可直接跳转到原始消息
- ✅ 查看每笔交易的详细时间
- ✅ 统计入款/下发总额

### 历史统计
```
/history
```
按日期显示历史账单记录。

### 成员账单
```
/member @成员
```
**示例：**
```
/member @张三
```
查看指定成员的详细账单。

### 月度报表
```
/month 2024-01
```
查看指定月份的收支报表。

### 导出账单
```
/export 2024-01
```
导出指定月份的账单数据（CSV 格式）。

## 🔧 管理员指令

### 日切时间设置
```
/setcutoff HH:mm
```
**示例：**
```
/setcutoff 03:00
```

### 汇率设置
```
/setrate <汇率>
```
**示例：**
```
/setrate 1.2
```
设置群组记账的本位币汇率。

### 手续费率设置
```
/setfee <费率>
```
**示例：**
```
/setfee 0.01
```
设置交易手续费率（1%）。

## ❓ 常见问题

### Q1: 如何知道我是标记员还是操作员？
**A:** 发送 `/标记员列表` 和 `/操作员列表` 查看。

### Q2: 为什么我不能直接记账？
**A:** 你需要是操作员才能使用 `金额 备注` 格式记账。请联系管理员设置你为操作员。

### Q3: 标记员和操作员有什么区别？
**A:** 
- **标记员**：只能使用 `@成员 金额 备注` 格式，用于群组 AA 记账
- **操作员**：可以直接使用 `金额 备注` 格式，也可以确认入账/出账

### Q4: 如何设置管理员？
**A:** 需要在配置文件中添加管理员 Telegram ID，或在群组的 bot 配置中设置。

### Q5: 记错账了怎么办？
**A:** 
1. 删除原消息，机器人会自动同步删除
2. 使用 `/edit` 命令修改
3. 联系操作员或管理员修正
